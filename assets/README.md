# Sistema de Gestión PQRS - Transporte HB

Aplicación web completa para la gestión de reportes PQRS (Peticiones, Quejas, Reclamos y Sugerencias) orientada a empresas de transporte con flota vehicular, utilizando Firebase como backend.

## Características

- **Formulario Público**: Interface para que los usuarios registren PQRS
- **Panel de Administración**: Interface para gestión interna de reportes
- **Autenticación**: Sistema seguro de acceso para administradores
- **Base de Datos en Tiempo Real**: Firebase Realtime Database
- **Diseño Responsive**: Compatible con dispositivos móviles y desktop
- **Exportación de Datos**: Generación de reportes en CSV
- **Filtros Avanzados**: Búsqueda y filtrado por múltiples criterios
